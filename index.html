Note: if not ordered properly, some formats might not work (e.g. using spoiler before code block, negating the spoiler)

Short Time (t):

HH:MM (AM/PM)

Long Time (T):

HH:MM:SS (AM/PM)

Short Date (d):

MM/DD/YYYY

Long Date (D):

Mo DD, YYYY

Full Date and Time (f):

Mo DD, YYYY HH:MM (AM/PM)

FD&T w/ Day (F):

Day, Mo DD, YYYY HH:MM (AM/PM)

Relative (R):

# (Unit) Ago
